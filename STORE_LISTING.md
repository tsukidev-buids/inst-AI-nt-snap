## Detailed Description

Inst-AI-nt Snap captures any web page's content in one click — text, images, and metadata — and stores it locally for research, studying, or AI-assisted work.

⚡ CAPTURE
• Snap full pages or highlight specific text
• Right-click context menu for quick capture
• Smart extraction detects articles, recipes, products, and code
• Deep capture scrolls lazy-loading pages automatically

📚 ORGANIZE
• Build a searchable clip library
• Create research projects and assign sources
• Tag quotes with one click
• Full-text search across all clips

📎 AUTO-CITE
• Generates citations automatically from page metadata
• APA, MLA, Harvard, and Chicago formats
• Switch styles instantly — all citations update
• Export a formatted bibliography for your paper

💾 EXPORT FOR AI
• Download AI-ready markdown prompts with your sources included
• Preset modes: Outline, Summarize, For & Against, Paraphrase, Introduction
• Upload to ChatGPT, Gemini, Copilot, or any AI
• Citations included — AI references your sources properly

🔒 PRIVACY FIRST
• All data stored locally — nothing leaves your browser
• No accounts, no cloud, no tracking
• AI features only activate if YOU add your own API key

FREE TIER

• 1 bibliography export per day
• Full library, search, and citation features
